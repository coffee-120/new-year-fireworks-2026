# 2026 Fireworks - 新年烟花特效

一个基于Three.js的3D烟花特效项目，专为2026新年设计。包含精美的烟花动画、音效系统和字幕功能。

## 项目特色

- 🎆 **逼真的3D烟花效果** - 使用Three.js实现的高质量烟花动画
- 🔊 **沉浸式音效系统** - 包含发射、爆炸、环境音效等
- 📝 **智能字幕功能** - 支持SRT格式字幕同步显示
- 🎨 **高级配色方案** - 金属质感+稀有化学色的专业配色
- 📱 **响应式设计** - 适配各种屏幕尺寸
- ⚡ **性能优化** - 支持大量粒子同时渲染

## 技术栈

- **Three.js** - 3D图形渲染引擎
- **Web Audio API** - 音频处理和播放
- **HTML5 Canvas** - 图形渲染
- **CSS3** - 样式和动画

直接打开 `index.html` 文件即可在浏览器中运行。

## 功能说明

### 烟花系统

- **多种烟花类型**：普通烟花、巨型烟花、超级烟花
- **物理模拟**：重力、阻力、爆炸冲击波
- **颜色变化**：随机颜色生成和渐变效果

### 音效系统

- **音效分类**：发射音效、爆炸音效、环境音效
- **音量控制**：支持烟花、背景音乐、语音音量独立调节
- **字幕同步**：自动同步显示SRT格式字幕

### 字幕功能

- **格式支持**：标准SRT字幕格式
- **双语显示**：支持中英文双语字幕
- **自动同步**：根据音频时间轴自动显示/隐藏字幕

## 配置选项

项目支持通过URL参数进行配置：

- `?debug=0` - 禁用调试模式
- `?overlay=0` - 隐藏调试信息覆盖层
- `?autostart=1` - 自动开始播放

## 开发说明

### 代码结构

项目采用模块化设计，主要包含以下模块：

- **配置系统** (`CONFIG`) - 管理项目配置参数
- **调试系统** (`DEBUG`) - 调试和日志功能
- **音效管理器** (`soundManager`) - 音频播放和控制
- **烟花引擎** - 烟花粒子和动画逻辑
- **字幕系统** - 字幕解析和显示

### 自定义开发

如需自定义烟花效果，可以修改以下配置：

```javascript
// 烟花参数配置
CONFIG.fireworks = {
  rocketsPerSecond: 3.2,    // 每秒发射火箭数量
  rocketBurstMin: 260,       // 最小爆炸粒子数
  rocketBurstMax: 520,       // 最大爆炸粒子数
  speedScale: 0.4,           // 速度缩放
  gravity: 5.2,             // 重力系数
  friction: 0.985            // 摩擦力
};

// 视觉效果配置
CONFIG.visuals = {
  bloom: {
    strength: 0.85,          // 光晕强度
    radius: 0.68,             // 光晕半径
    threshold: 0.22           // 光晕阈值
  }
};
```
